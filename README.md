# 🚀 TaskFlow Pro

A Full Stack To-Do List Management System developed using Spring Boot, PostgreSQL, HTML, CSS and JavaScript.

## Features

- Add Tasks
- Delete Tasks
- Mark as Complete
- Task Priority
- Search Tasks
- Dashboard Statistics
- PostgreSQL Database Integration
- Responsive UI

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- HTML
- CSS
- JavaScript
- Maven

## Project Structure

Frontend
↓
REST API
↓
Spring Boot
↓
Service Layer
↓
Repository Layer
↓
PostgreSQL

## How to Run

1. Install Java 25
2. Install PostgreSQL
3. Create database

```sql
CREATE DATABASE todolist_db;
```

4. Update application.properties

```properties
spring.datasource.username=postgres
spring.datasource.password=YOUR_PASSWORD
```

5. Run

```
TodolistApplication.java
```

6. Open

```
http://localhost:8080/index.html
```
